# reference type

## keyword để khai báo reference type

- class
- interface
- deregate
- record

## các built-in type

- object
- string
- dynamic

# **class**
- class được kế thừa từ 1 class, 1 class + 1 interface hoặc 1 class + nhiều interface
- class có thể nested trong class parent
